## 编程语言

### 语言标记

语言标记是一种简单的描述语言的方法，由语言识别符和标签用冒号连接而成。如果没有标签，冒号可以省略。

语言标记区分大小写。

#### C++ 语言

识别符为 `c++` 或含 `c++` 前缀的串。

识别符会用于语言标准，即传递给 g++ 的 `-std=` 选项，但 `c++` `c++20` `c++23` `c++26` 会分别被翻译为 `c++14` `c++2a` `c++2b` `c++2c`。

标签：

- `Ox` 启用对应等级优化，例如 `O2` `Ofast`。这将被直接传递给编译器，不加额外检查。
- `Makefile` 详见 [使用 Makefile](#使用-makefile)。

标签对大小写敏感。

### 使用 Makefile

C++ 语言指定标签 `Makefile` 时，将使用 Makefile 而忽略其它标签。

你需要按照下表编写编译目标，否则会产生编译错误。编译目标同时也是源程序名称和编译得到的可执行文件的名称。

| 模块 | 编译目标 |
| :-: | :-: |
| 选手程序 | program |
| 校验器 | checker |
| 交互库 | interactor |

例如，编译校验器和交互库的 Makefile 可以这样写：

```makefile
checker:
    g++ checker.cpp -o checker -O2
interactor:
    g++ interactor.cpp -o interactor -O2
```

对应语言选项为 `c++:Makefile`。
